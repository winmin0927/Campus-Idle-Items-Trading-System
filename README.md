功能描述（业务功能不少于6个，至少含一个定时统计功能）
1、物品发布与搜索：用户发布自己的闲置物品，包括书籍、电子设备、家具等。其他用户可以搜索并浏览这些物品。
2、用户身份验证与权限管理：用户注册和登录功能；不同用户角色（如买家、卖家、管理员）应该有不同的权限。
3、订单统计功能：定期统计交易数据，例如每周、每月或每学期。
4、物品详情页面：物品描述、价格、照片和联系方式。
5、交易管理：支持交易的管理，包括订单状态跟踪、付款确认和物品配送。卖家和买家查看交易历史。
6、通知与提醒：发送通知和提醒，例如物品发布成功、交易状态更新或定时统计报告。

主业务流程描述（不少于2个）
1、物品发布流程：
-用户登录系统。
-点击“发布物品”按钮。
-输入物品信息，包括名称、描述、价格和照片。
-确认发布。
2、物品购买流程：
-用户登录系统。
-浏览物品列表，选择感兴趣的物品。
-点击“购买”按钮。
-输入付款信息。
-确认购买。
-系统将订单状态更新为“已购买”。
